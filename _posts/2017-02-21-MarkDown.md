---
##1. 标题
  ```
  # 一级标题 ## 二级标题 ### 三级标题 #### 四级标题 ##### 五级标题 ###### 六级标题
  ```
##2. 列表
```
- 无序列表  1. 有序列表
```
##3. 引用
 文章中引用了别人的文章可以使用 “引用”
 
```
> 文章内容
```

```
>> 二级引用
```
##4. 粗体斜体
```
*斜体字*  **粗体字**
```
##5. 链接
```
[链接文字](url链接地址)
```
##6. 图片
```
![图片名字](url地址)
```
![图片名字](http://img3.douban.com/mpic/s1108264.jpg)
本地图片可以直接拖动图片到编辑框中
##7. 分割线和删除线
```
---  *** 分割线  
```
---
```
~~删除文字~~
```
~~Mistaken text.~~
